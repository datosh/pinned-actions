GitHub Pinned Actions

Download raw data (results.json.tar.gz)

GitHub recommends to pin an action to a full length commit SHA as it is currently the only way to use an action as an immutable release.

Still, only 3% of GitHub repositories follow this security best practice!

Why should you pin your GitHub actions?

  • Stable pipelines
  • Improved security

How-to secure your GitHub actions?

  • Frizbee to migrate
  • Dependabot to keep up to date
  • Minder to enforce policy
